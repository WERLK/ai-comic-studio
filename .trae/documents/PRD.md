# AI 漫剧制作应用 - 产品需求文档 (PRD)

## 1. 产品概述

AI 漫剧制作应用是一款利用人工智能技术，一键生成漫画和动态漫剧作品的创意工具。用户通过输入故事文字或上传素材，选择画风风格，即可自动生成包含场景、人物、分镜和对话的完整漫剧作品，并支持导出为静态漫画或动态视频。

### 目标用户
- 漫画创作者和爱好者
- 内容创作者和社交媒体博主
- 广告创意人员
- 教育内容制作者

### 核心价值
- **一键生成**：输入故事即可自动生成完整漫剧，无需手动编辑
- **双模式输出**：既可生成静态漫画页面，也可生成动态漫剧视频
- **灵活输入**：支持文字输入和素材上传两种方式
- **风格多样**：提供日系动漫、经典漫画、赛博朋克、写实等多种画风选择

---

## 2. 核心功能

### 2.1 功能模块

1. **创作首页 (/)**：故事输入、项目管理
2. **AI生成器 (/generator/:id)**：一键生成漫剧、漫画/视频双视图预览
3. **导出页面 (/preview/:id)**：导出设置、格式选择

### 2.2 页面详情

| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 创作首页 | 故事输入 | 多行文本框输入故事内容，设置项目名称 |
| 创作首页 | 素材上传 | 上传图片或视频作为创作素材 |
| 创作首页 | 生成设置 | 选择画风格式、分镜数量、角色数量 |
| 创作首页 | 项目列表 | 展示用户所有项目，支持删除、继续编辑 |
| AI生成器 | 进度展示 | 显示AI生成各阶段的进度（角色→场景→分镜→对话） |
| AI生成器 | 漫画视图 | 网格展示所有分镜，点击可查看详情 |
| AI生成器 | 视频预览 | 单帧预览，带播放控制和时间轴 |
| 导出页面 | 格式选择 | PNG静态图、PDF漫画册、MP4视频、GIF动画 |
| 导出页面 | 分辨率选择 | 720p、1080p、2K、4K |

---

## 3. 核心流程

### 3.1 主流程：一键生成漫剧

```
进入首页 → 输入故事/上传素材 → 选择风格和设置 → 点击生成
↓
AI自动分析故事 → 生成角色形象 → 生成场景背景 → 生成分镜布局 → 添加对话旁白
↓
生成完成 → 预览漫画/视频 → 导出作品
```

### 3.2 输入模式

**文字输入模式**：
1. 输入项目名称
2. 在文本框中输入故事内容
3. AI自动分析故事情节生成分镜

**素材上传模式**：
1. 输入项目名称
2. 上传已有的图片或视频
3. AI根据素材内容创作漫剧

## 4. 用户界面设计

### 4.1 设计风格

**美学方向：赛博朋克 × 日系漫画融合**

- 主色调：深邃紫 (#2D1B4E) 搭配霓虹粉 (#FF2E63) 和电光蓝 (#08F7FE)
- 强调色：荧光黄 (#F7F052) 用于高亮和 CTA 按钮
- 背景：深色系 (#0D0D0D, #1A1A2E) 配合网格纹理
- 日系漫画元素：速度线、网点背景、对话气泡

**字体选择**
- 标题：Orbitron (科技感) 或 Noto Sans SC Bold
- 正文：Noto Sans SC Regular
- 对话气泡：手写风格字体

**按钮风格**
- 主按钮：霓虹发光边框 + 渐变填充
- 次按钮：幽灵按钮，透明底色 + 发光边框
- 悬停效果：发光增强 + 轻微上浮

### 4.2 页面布局

**创作首页**：
- 桌面端：左右分栏布局（输入区60% + 项目列表40%）
- 移动端：垂直堆叠，输入区在上，项目列表在下

**AI生成器**：
- 全屏界面，无侧边栏
- 顶部：返回按钮 + 项目名称 + 状态标签
- 中部：漫画网格视图或视频预览区
- 底部（视频模式）：播放控制栏 + 时间轴

**导出页面**：
- 居中卡片式布局
- 格式选择网格
- 分辨率选择按钮组
- 导出预览信息
- 导出按钮

### 4.3 响应式设计

- 桌面优先 (1440px+ 完整体验)
- 平板适配 (768px-1439px 简化布局)
- 移动端适配 (320px-767px)
  - 汉堡菜单
  - 单列布局
  - 全屏模态框
  - 触摸友好的按钮尺寸

## 5. 技术需求

### 5.1 前端技术

- React 18 + TypeScript
- Tailwind CSS 3
- Vite 构建工具
- Zustand 状态管理
- Framer Motion 动画
- React Router 路由

### 5.2 AI 能力集成 (模拟)

由于当前为演示版本，AI生成功能为模拟实现：

```typescript
// 模拟AI生成流程
1. 分析故事文本 → 提取关键词和情节
2. 生成角色形象 → 使用 DiceBear API 模拟
3. 生成场景背景 → 使用 Unsplash 图片模拟
4. 生成分镜布局 → 根据分镜数量均分画面
5. 添加对话旁白 → 从故事文本中提取或生成
```

### 5.3 数据存储

- 项目数据：localStorage
- 生成的图片：URL外部资源
- 项目结构：
```typescript
interface Project {
  id: string;
  title: string;
  sourceContent: string;      // 用户输入的故事或素材描述
  sourceType: 'text' | 'upload';
  frames: Frame[];            // 生成的所有分镜
  status: 'draft' | 'generating' | 'completed';
  createdAt: string;
  updatedAt: string;
}
```

## 6. 导出功能

### 6.1 导出格式

| 格式 | 描述 | 用途 |
|------|------|------|
| PNG | 静态图片 | 单张漫画页面 |
| PDF | 漫画册 | 多页漫画电子书 |
| MP4 | 视频文件 | 动态漫剧视频 |
| GIF | 动画文件 | 社交媒体分享 |

### 6.2 分辨率选项

| 分辨率 | 尺寸 | 适用场景 |
|--------|------|----------|
| 720p | 1280×720 | 移动端预览 |
| 1080p | 1920×1080 | 主流视频平台 |
| 2K | 2560×1440 | 高清显示 |
| 4K | 3840×2160 | 专业制作 |
